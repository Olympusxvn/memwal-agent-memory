What is it?

MemWal Agent Memory lets autonomous agents remember, recall, prove, share, and monetize knowledge — without being locked to one app or losing context every session.

It wraps the official MemWal SDK (Walrus Memory). It does not fork Mysten’s stack. It adds:

  • Local-first speed and privacy (SQLite + vectors)
  • Controlled promotion to Walrus (redaction + quality gates)
  • Sui Move marketplace, bounties, and royalties
  • MCP for Cursor, Claude, OpenClaw, or any MCP client

Display name MemWal++ in some UI; repo scope: memwal-agent-memory / @memwalpp/*.

How it works

  1. Write locally — SQLite (fast, private)
  2. Gate — redact + quality scorer
  3. Promote — MemWal remember → SEAL → Walrus blob
  4. Recall — hybrid local + durable semantic search
  5. Trade / bounty — Move + Walrus-backed fulfillment
