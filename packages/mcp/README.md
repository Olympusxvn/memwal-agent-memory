# @memwalpp/mcp

MCP server — **project memory for Cursor and Claude Desktop**.

- **Pro Local:** SQLite, no API keys — `remember`, `recall`, `search`
- **+ Walrus Sync:** optional MemWal → Walrus via env config

## Install (from monorepo)

```bash
pnpm install && pnpm mcp:build
node packages/mcp/dist/cli.js --transport stdio
```
